## チュートリアル

## 始め方
```sh
#初期化
npm init -y

# TypeScriptをローカルにインストール
npm i --save-dev typescript
```

## コマンド集
```sh
# コンパイル
npx tsc [ファイルパス]

# コンパイル後に生成されたJSファイルを実行
node [ファイルパス]
```
